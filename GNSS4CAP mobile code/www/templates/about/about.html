EGNSS4CAP is a mobile phone application for farmers in the European Union to digitalise procedures surrounding their reporting requirements under the current and post-2020 Common Agricultural Policy (CAP) reform.

This application enables farmers to provide geotagged photos to support and complement a Copernicus Sentinel-based monitoring approach for CAP.

It is using EGNSS (EGNOS and Galileo) differentiators such as OS-NMA and dual-frequency capabilities to reduce errors and duplication, as well as improve performance and trustworthiness.

Galileo Signal Check: green light if this device is able to use the Galileo signals.

Dual Frequency Signal Check: green light if this device is able to track GNSS dual-frequency (L1/L5 for GPS and E1/E5 for Galileo).

EGNOS Signal Check: green light if this device is able to work with EGNOS/SBAS satellites.

Galileo Navigation Message Check: green light if the device firmware is accessing to the GNSS raw data in general and Galileo Navigation message in particular, being capable to benefit from Galileo Open Service Navigation Message Authentication (OS-NMA).

The project team has worked closely with the paying agencies responsible for administering subsidies for farmers and JRC to ensure that the minimum user requirements will be met.
